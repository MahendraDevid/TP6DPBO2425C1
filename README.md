# Design Program
Program ini adalah game Flappy Bird (Saya ubah jadi FLapyy Car) yang dibuat menggunakan Java Swing. Game ini berjalan dengan bantuan timer untuk mengatur pergerakan dan jalannya permainan. Konsep OOP digunakan dengan membagi program ke dalam beberapa class, antara lain:

- Player: mengatur karakter burung (posisi dan lompatan).
- Pipe: mengatur rintangan yang muncul dalam permainan.
- Logic: mengatur jalannya permainan seperti pergerakan, skor, dan kondisi kalah.
- View: menampilkan tampilan game ke layar.
- App: membuat jendela game dan menjalankan program.
- SoundManager: mengatur suara dalam game, seperti suara lompat dan suara lainnya.

# Alur Program
1. Pemain akan ditunjukkan dalam main menu yaitu tombol play untuk memainkan permainan / exit untuk keluar dari game
2. Pemain akan memainkan game tersebut dengan menggunakan tombol space untuk melompat, dengan pipe yang selalu muncul secara random
3. Jika Pemain Menabrak Pipe atau jatuh kebawah, maka Game Over
4. Pemain bisa tekan tombol R untuk Restart Permainan
